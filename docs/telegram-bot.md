# Telegram Bot 使用说明

## 前置条件
- 拥有 Telegram 账号
- 已创建 Bot（通过 @BotFather）

## 配置步骤
1. 获取 Bot Token
2. 在 WebUI → 系统设置 → Telegram Bot 设置中配置 Token
3. 启用 Bot 并测试连接
4. 用户在个人中心获取绑定码
5. 在 Telegram 中发送 `/bind 绑定码`

## 常用命令
- `/start`：显示帮助
- `/search 关键词`：搜索书籍
- `/recent`：最近新增
- `/library`：书库浏览
- `/download <书籍ID>`：下载书籍
- `/info <书籍ID>`：书籍详情
- `/read <书籍ID>`：TXT 在线阅读
- `/favorites`：收藏列表

## TXT 在线阅读
- Telegram 内按字节分页
- 支持上一页/下一页
- 详情页提供“开始/继续阅读”按钮

## 小提示
- 书名、作者可点击跳转对应搜索
- 书库列表支持筛选与排序
